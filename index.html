About

Kepler Std. Bold Italic

Fractal Generator is a web demo that generates fractal patterns from the letter ‘f’ using Javascript Canvas. It creates 30 instances of the same letter ‘f’, arranged in 3 unique circular formations. Each formation is assigned different properties including letter weight, angle, and radius — allowing for an infinite number of pattern possibilities.
